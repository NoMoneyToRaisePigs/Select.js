Select Component

  • simple

    Initializing a select by just passing the element ID and your input source:

    var select = new gf_select('id', 'source');

  • flexible

    customize your own data.

    PropertyUsage
    multitrue to make it a multi-select
    searchabledo you need to search ?
    datagroups/options for you select

  • light

    there is no css, js is only 10K!

  • written in plain JS and CSS

    no ployfill, no icon, no svg, no picture, pure JS and CSS

  • cross browsers supported
